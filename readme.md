# How to use

## 1. Download LZMA SDK

* downlaod site: https://www.7-zip.org/sdk.html
* unzip lzma1900.7z
* Go to `\lzma1900\CPP\7zip\UI`

## 2. Clone repo

* this repo clone in `\lzma1900 (2)\CPP\7zip\UI`
```
git clone https://github.com/lucidmaj7/lm77z.git
```

## 3. Open Solution

* open lm77z.sln 

# Environment

* Visual Studio 2013
* LMZA SDK 1900 ver
